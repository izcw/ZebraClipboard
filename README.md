# ZebraClipboard
### 斑马在线剪贴板，快速分享文本数据



## 前台
### Frontend/document

```bash
pnpm install  # 安装依赖
pnpm docs:dev # 启动
```

在Frontend/document/.vitepress/utils/config.js中切换请求接口


## 后台

### Frontend/program

```bash
pnpm install # 安装依赖
pnpm dev # 启动
```

在Frontend/program/src/utils/config.js中切换请求接口


## 后端

### Backend

```bash
pnpm install # 安装依赖
pnpm dev # 启动
```
